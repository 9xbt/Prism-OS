### PrismBinary
A minimal library containing several loaders for different binary formats, like compression, markup languages, and more to come eventualy.
Here is a full list of the currently supported formats:

| Format Name | Working? |
|-------------|----------|
| ELF         | No?      |
| INI         | Yes      |
| LZW         | Yes      |
| ZLib        | Yes      |
| Tar         | Yes      |
